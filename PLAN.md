# 烽享财税全新网站设计制作计划

## 一、品牌视觉定义

| 项目 | 内容 |
|------|------|
| **品牌名** | 烽享财税 — 一站式企业财税综合服务平台 |
| **公司全称** | 广东烽享联动信息科技有限公司 |

| **辅助色** | 深灰 `#212121`、白 `#FFFFFF`、浅灰 `#F5F5F5` |

## 二、已确认的技术决策

| 项目 | 决策 |
|------|------|
| **域名** | 继续使用 `www.fengxiangld.cn` |
| **部署方式** | 静态HTML托管（阿里云OSS/腾讯云COS） |
| **技术栈** | 纯静态 HTML5 + CSS3 + Vanilla JavaScript |
| **多语言** | 仅简体中文 |
| **移动端** | 纯响应式设计（一套代码适配PC+手机+平板） |
| **内容复用** | 保留旧站公司介绍/业务描述/联系方式，迁移历史文章/案例 |

## 三、功能模块优先级（已排序）

### 首期上线（Phase 1）

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0 | **在线留言/表单提交** | 客户获取核心功能，联系页必带 |
| P0 | **微信公众号/视频号入口** | 简单实现（二维码+链接），引流关键通道 |
| P1 | **预约咨询/在线通话** | 电话直拨按钮 + 微信二维码引导 |

### 二期上线（Phase 2）

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P2 | **SEO博客/文章系统** | 静态站做文章需模板化，后续扩展 |

## 四、网站结构规划（7个页面）

### 4.1 首页（index.html）

```
[Header: Logo + 导航 + 电话按钮]
├── Hero Banner（大标题 + 副标题 + CTA按钮 + 蜂巢装饰）
├── 核心服务（4张卡片：灵活用工合规 / 数电发票 / 工商财税 / 税务咨询）
├── 数据统计（12年+经验 / 服务企业数 / 客户满意度）
├── 烽享优势（6个卖点：更安全 / 更智能 / 更高效 / 更多元 / 更专业 / 更放心）
├── 适用行业（物流/教育/出行/直播/医疗等行业图标）
├── CTA区（免费咨询入口）
└── Footer（联系方式 + 备案信息 + 公众号/视频号二维码）
```

### 4.2 关于我们（about.html）
- 公司简介（沿旧站文案）
- 企业价值观：分享 · 专注 · 审慎
- 发展历程（时间轴）
- 团队展示

### 4.3 业务板块（services.html + 4个子页）
| 服务 | 描述 |
|------|------|
| 灵活用工合规 | 灵活用工平台，多税地解决方案 |
| 数电发票 | 兼容数电发票和纸质发票 |
| 工商财税 | 公司注册、代理记账、商标注册、软著申请 |
| 税务咨询 | 专业涉税咨询服务 |

### 4.4 HiveMind（hivemind.html）
- AI产品品牌独立页面

### 4.5 案例中心（cases.html）
- 客户案例网格展示（迁移旧站历史案例）

### 4.6 新闻动态（news.html）
- 行业资讯 + 政策解读（迁移旧站历史文章）

### 4.7 联系我们（contact.html）
- 在线留言表单（P0首期）
- 电话：152-1760-9018
- 微信客服二维码
- 公众号/视频号入口
